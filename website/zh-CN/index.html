Pluck

让熟悉的 Git 操作成为肌肉记忆。

一款专注于 macOS 的 Git 客户端，操作方式参考 JetBrains 系列 IDE 的 Git 工作流。

熟悉的工作流，完整的上下文。

在一个专注的工作区中查看提交历史、工作区变更和分支操作。处理日常 Git 工作时，始终保留每项变更背后的上下文。

围绕仓库构建的工作流

可视化历史

在清晰的历史记录中追踪分支与提交，无需打断当前流程即可打开并查看任意变更。

为审阅而生的 Diff

查看已暂存和未暂存的变更，同时保留相关文件与提交上下文，方便理解和审阅。

交互式 Rebase

通过引导式交互 rebase 流程，对提交进行重新排序、合并、编辑或丢弃。

安全的分支操作

创建、切换、重命名或删除分支前，先看清仓库当前状态再执行操作。

Pluck 的构建原则

熟悉的设计

常用 Git 操作沿用开发者在 JetBrains 系列 IDE 中熟悉的交互方式。

Git 作为底层

Pluck 通过 Git CLI 执行仓库操作，让 Git 本身始终作为事实来源。

专注于工作本身

界面始终围绕理解代码变更和完成下一步仓库操作展开。

通过 DMG 安装

  1. 从最新 Release 下载 DMG 文件。
  2. 打开 DMG，将 Pluck 拖入“应用程序”文件夹。
  3. 如果 macOS 阻止打开应用，请打开“终端”并运行：
xattr -dr com.apple.quarantine /Applications/Pluck.app
