# Pluck

A native, keyboard-friendly git client for macOS. Built with Tauri 2 + Vue 3.

## Status

Personal-use alpha. macOS arm64 only for now; Windows support planned.

## Run from source

```bash
pnpm install
pnpm tauri dev
```

## Tech stack

- Tauri 2 (Rust backend)
- Vue 3 + Pinia + Vite + TailwindCSS 4
- TypeScript
